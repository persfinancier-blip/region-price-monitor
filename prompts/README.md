# prompts/

Промпты для воркера (Claude Code на GitHub Actions). Пишет их Cowork (см. `COWORK.md` в корне).

## Правила

- **Активные** (написаны, ещё не выполнены) — в корне `prompts/`.
- **Выполненные и смёрженные** — воркер сам переносит в `prompts/_done/` в том же PR.
- Именование: `prompt-NN-краткое-имя.md` (продукт), `prompt-ops-NN-…` (инфраструктура).
- `_done/` — не свалка, а **образец стиля**: смотри туда, если сомневаешься в формате.

## Активные

- `prompt-00-spike.md` — спайк осуществимости, гейт GO/NO-GO (Фаза −1).
